3. Advantages of Pandas

  • it can read or write data from many different data formats like int, float, double etc.
  • it supports reshaping of the data in different forms
  • it can calculate data in all possible ways data is organized that is across the rows and down the group
  • it supports visualization by integrating matpoltlib and seaborn libraries

4. What is data structure

Data Structure refer to specialized way of storing data so as to apply specific type of functionality on them.

A data structure is a particular way of storing and organizing data in a computer to suit a specific purpose so it can be accesed and worked with in appropriate ways.

5. Series Data Stuructures

Series is panda’s data strucutre which represents 1 dimensional array - like object containing an array of data and an associated array of data-lables called its index

5.1) Creating empty Series Object

Use Series() with no parameters

Syntax : <Series_Object = pandas.Series()

5.2) Creating Non - Empty Series object

Syntax : Series_object = pandas.Series(data, index = idx)

here idx = variable

We can specify data type in following parts

  1. A python Sequence
  2. An nd Array
  3. A python dictionary
  4. A scalar value

5.2.1) Specify data as Python Sequence

Syntax : <Series object> = Series(<any python sequence>)
